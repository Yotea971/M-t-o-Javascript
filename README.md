# Météo en JavaScript

Application météo simple en **JavaScript** qui permet de saisir une ville et d’afficher la température et d’autres informations météorologiques en temps réel via une API.

##  Fonctionnalités

- Saisie d’une ville dans un champ de recherche.  
- Validation au clic sur le bouton **Rechercher**.  
- Validation également possible avec la touche **Entrée** sur le champ de saisie.  
- Affichage de la température, de la description du temps, et éventuellement d’autres données (humidité, vent, etc.).  

##  Technologies utilisées

- HTML  
- CSS  
- JavaScript  
- API de météo (API utilisé OpenWeatherMap)
